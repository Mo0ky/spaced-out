SpacedOut

A three dimensional CSS3 Transform Sidebar

SpacedOut uses CSS transforms to create a space efficient menu. The original menu for this design is at atmospherejs.com, props to the studio who built the site. Decided to build SpacedOut as a standalone library for future personal use.

Does not use jQuery. Usage and Documentation found in the github repository
